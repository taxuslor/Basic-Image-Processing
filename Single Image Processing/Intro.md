# Some Basic Techniques for Singe Image Processing

- reformat.py:
  - convert a jpg to png
  - save (all) the frames of gifs
- gray.py:
  - graying
  - binaryzation
  - histogram equalization
  - local contrast enhancement
- size.py
  - resize
  - crop
- filter.py
  - smooth
  - sharpen
  - edge detection
- color.py
  - adjust brightness
  - adjust contrast
  - adjust saturation
- enhance.py
  - denoising
  - enhance details
- perspective.py
  - transform perspective
